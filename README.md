# KeenKeeper

KeenKeeper is a responsive friendship tracking web app that helps users stay connected with the people who matter most. Users can view friend profiles, log quick check-ins, review a full interaction timeline, and analyze communication habits through a pie chart.

## Live Link
https://b13-a07-keen-keeper-beta.vercel.app/

## GitHub Repository
https://github.com/meheroon/B13A07-Keen-keepe

## Technologies Used
- Next.js
- React
- Tailwind CSS
- React Icons
- React Hot Toast
- Recharts

## Key Features
- Dynamic friend cards loaded from a JSON file with status-based UI
- Quick check-in actions that instantly create timeline entries with toast notifications
- Friendship analytics page with a Recharts pie chart and timeline filtering

## How to Run Locally
```bash
npm install
npm run dev
```

## Routes
- `/` → Home page
- `/friends/[id]` → Friend details page
- `/timeline` → Interaction timeline page
- `/stats` → Friendship analytics page
